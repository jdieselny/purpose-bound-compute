SYNTH-DESK(1) — Purpose-Bound Compute

NAME
synth-desk — companion terminal (HTTP of AI): define L1–L4 context, enroll thumbprint, multi-provider chat

SYNOPSIS

node demo/shared/bridge/bridge.js
open demo/synth-desk/index.html

DESCRIPTION
Synth Desk is the public demo surface for companions: portable synth bodies that share identity across model providers. Private Ed25519 keys never enter the browser; provider API keys stay in browser localStorage only.
