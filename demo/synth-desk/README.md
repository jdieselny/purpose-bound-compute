# Synth Desk — Companions (HTTP of AI)

**COSA-MAP:** synth edge · context stack L1–L4 · enrollment · simulated intent router → **FILLED / PARTIAL**

Create and register **companions**: portable synth bodies defined by a four-layer context stack, enrolled by thumbprint, chatting across model providers.

## Quickstart

```bash
# from repo root
node demo/shared/bridge/bridge.js
```

Open `demo/synth-desk/index.html` in a browser.

1. Complete **first-time setup** (overlay) — set at least one API key (browser localStorage only).
2. Fill **L1–L4** context stack (or load the companion preset).
3. Click **ENROLL** to register thumbprint in local truthroot.
4. Enable tools and chat.

## Out of scope here

Workers, shift discipline, inbox routing, and offline verify live in [PBC Shift](../pbc-shift/README.md).

## Security

- Private keys never enter the browser.
- Local demo bridge binds `127.0.0.1` only.
- Integrity of a receipt ≠ physical truth of the world.
